#  EXCALIBUR: Sovereign Staff Product Architect Ecosystem

**EXCALIBUR** is an industrial-grade engineering ecosystem designed to project the technical and strategic authority of a **Staff Product Architect**. Built on Next.js 16 and a sovereign data orchestration pipeline, it merges high-fidelity visual design with deep RAG (Retrieval-Augmented Generation) intelligence.

## 🛡️ Core Pillars

### 1. Sovereign Architecture
The platform utilizes an **Iframe Sovereignty** pattern. The original, certified design resides in an isolated layer (`public/original/index_clean.html`), protecting its 1:1 visual parity from Next.js/Tailwind side-effects while allowing dynamic injection of forensic evidence.

### 2. Forensic RAG Intelligence
Integrated with a custom **AI Cortex (Titan Agent)**, the system performs real-time retrieval from a database of 24 technical artifacts (PDFs). Every milestone in the career is backed by verified data, presented using the **PAR Method (Problem, Action, Result)** for maximum executive clarity.

### 3. A11y Governance (ARC)
The **Accessibility Restoration Core (ARC)** HUD performs live DOM auditing on the sovereign layer, ensuring compliance with **WCAG 2.2 AAA** standards. It includes specialized modules for Keyboard Navigation, Landmark Structure, and Heading Hierarchy.

### 4. Design System Lockdown
Standardized entirely under **Roboto Flex**, the ecosystem utilizes variable font axes to maintain hierarchy and readability across all sovereign and orchestration layers.

## 🛠️ The Problem
Maintaining a pixel-perfect, certified portfolio while adding modern dynamic capabilities (AI, real-time auditing) usually leads to "style bleeding" and performance degradation. Manual updates of career evidence often result in inconsistent narratives and technical debt.

## 🚀 The Solution (Architecture)
EXCALIBUR solves this through a **Master Orchestrator Pipeline**. It separates the **Source of Truth (HTML/CSS)** from the **Data Layer (JSON/RAG)**. During the build process, the orchestrator physically purges legacy assets and injects verified evidence into the SSOT layer, ensuring the final artifact is both technically honest and visually supreme.
