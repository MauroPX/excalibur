#  Ecosistema EXCALIBUR: Design System & Visual Language

**EXCALIBUR** utiliza un sistema de diseño propio basado en los principios de estética de iOS 26 "Liquid Glass".

## 🖋️ Soberanía Tipográfica: Roboto Flex
Se ha implementado **Roboto Flex** como fuente única para garantizar una accesibilidad universal sin comprometer la jerarquía visual.

### Ejes de Variación (Variable Font)
- **Títulos (h1-h6, Logo)**: `wght: 1000`, `wdth: 120`, `opsz: 48`. Proyecta fuerza y autoridad técnica.
- **Cuerpo de Texto**: `wght: 400`, `wdth: 100`, `opsz: 14`. Optimizado para lectura de larga duración con interlineado de `1.45`.

## 💎 Liquid Glass Engine
El motor visual utiliza una arquitectura de capas con desenfoque gaussiano profundo y radios de squircle calibrados.

- **Fondo**: `rgba(10, 10, 10, 0.9)`.
- **Efecto Blur**: `60px saturate(210%)`.
- **Bordes**: `1px solid rgba(255, 255, 255, 0.18)` con reflejo líquido.

## 🧱 Componentes Atómicos (Staff Atoms)
- **LiquidFAB**: Botones flotantes de 56px con z-index de sistema para visibilidad garantizada.
- **PAR Cards**: Tarjetas de evidencia técnica estructuradas bajo el método Problema, Acción, Resultado.
